Diferencias De HTML

Diferencias de html 4 y html 5
Caracteristicas HTML 4 HTML 5
Nuevos elementos semánticos No hay elementos semánticos definidos Introduce nuevos elementos semánticos como header nav,section,article,etc.
Multimedia Dependencia de Flash para multimedia Introduce elementos multimedia nativos como audio y video
Formularios Formularios menos avanzados Introduce nuevos tipos de entrada y atributos para formularios , facilitando la validación y la experiencia del usuario
Almacenamiento web No hay almacenimiento web Introduce el almacenimiento local y de sesión (localStorage y sessionStorage)
Arrastrar y soltar No compatible Introduce API de arrastrar y soltar (Drag and Drop)
Integración con CSS3 Trabaja junto con CSS2 Permite una mejor integración con CSS3 para aplicar estilos y efectos avanzados
